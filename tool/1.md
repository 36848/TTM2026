# how to use
only  need change Latitude,Longitude,zoom,half_range

## explain use table
|  Zoom Level |  Display Level | Usage |
|----------------------|-------------------------|--------------|
| 10 – 12 | 城市级 / City Level | 大区域概览，适合宏观规划 / Overview of large area, suitable for macro planning |
| 14 – 16 | 街道级 / Street Level | 常规地图查看，能看清主要道路 / Regular map view, main roads visible |
| 17 – 18 | 高精度 / High Precision | 建筑物轮廓、细部道路详情 / Building outlines and detailed roads |
| 19 | 超高精度 / Ultra High Precision | 极度清晰（注意：部分偏远地区可能无此级别数据） / Extremely clear (some remote areas may not have this level) |

| half_range |  Tiles Downloaded |  Map Area Description |
|------------|-------------------------------|-------------------------------------|
| 1 | 3×3 = 9 张 / 3×3 = 9 tiles | 极小区域（测试用） / Very small area (for testing) |
| 2（默认） | 5×5 = 25 张 / 5×5 = 25 tiles | 中等区域 / Medium area |
| 3 | 7×7 = 49 张 / 7×7 = 49 tiles | 稍大区域 / Slightly larger area |
| 5 | 11×11 = 121 张 / 11×11 = 121 tiles | 大范围区域 / Large area |
| 10 | 21×21 = 441 张 / 21×21 = 441 tiles | 非常大范围（拼接压力较大） / Very large area (high stitching load) |
